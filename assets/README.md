This folder contains all images, icons, and fonts for the PawGuard app.

## Required Files

### 1. App Icon (icon.png)
- **Size:** 1024x1024 pixels
- **Format:** PNG
- **Background:** Transparent or solid color
- **Design:** Your PawGuard logo
- **Color scheme:** Warm colors (orange, coral) matching the app theme

### 2. Splash Screen (splash.png)
- **Size:** 2048x2732 pixels (portrait)
- **Format:** PNG
- **Background color:** #FF8C61 (warm orange/coral)
- **Design:** Logo in center with optional tagline
- **Aspect ratio:** 3:4

### 3. Adaptive Icon (adaptive-icon.png)
- **Size:** 1024x1024 pixels
- **Format:** PNG
- **Safe zone:** Inner 66% circle for icon (outer 33% may be cropped on Android)
- **Background:** Should work on any background color

### 4. Notification Icon (notification-icon.png)
- **Size:** 256x256 pixels
- **Format:** PNG
- **Color:** White icon on transparent background
- **Design:** Simple, recognizable silhouette

### 5. Favicon (favicon.png)
- **Size:** 48x48 pixels
- **Format:** PNG
- **Design:** Simplified version of app icon

## Design Guidelines

### Color Palette (Warm & Friendly)
- **Primary:** #FF8C61 (Coral Orange)
- **Secondary:** #6BADC6 (Soft Blue)
- **Accent:** #FFD93D (Warm Yellow)
- **Success:** #4CAF50 (Green)
- **Background:** #F8F8F8 (Off-white)

### Style
- **Friendly and approachable**
- **Pet-themed** (paw prints, dog/cat silhouettes)
- **Modern and clean**
- **Professional yet warm**

## Testing Your Icons

After adding real icons:
1. Clear Expo cache: `expo start -c`
2. Test on real devices (iOS and Android)
3. Check how they appear in:
   - App launcher
   - Splash screen
   - Notifications
   - Settings menu
